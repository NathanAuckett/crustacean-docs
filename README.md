# Documentation
This documentation contains useful information in using Game Maker and the tools created specifically for this project.

Use the side-bar on the left to browse or search the documentation.

## Common terms <!-- {docsify-ignore} -->
- "String" = A piece(*string*) of text, can contain any character.
- "Variable" = A named value, like in algebra. Example:
    ```gml
    a = 2; //This is an integer value
    fruit = "grape"; //This is a string value
    youAreACoolGuy = true; //A boolean value
    collectionOfNumbers = [0, 1, 2, 3, 4]; //An array
    ```
